A portfolio should feel polished, but it should also prove that the developer can build maintainable systems.

## Let the Server Do Server Work

React Server Components are a natural fit for project pages because the project list is mostly read-only content. Fetching it on the server reduces client JavaScript and keeps the page simple.

## Validate Every Boundary

Forms and API routes are public boundaries. Zod gives the code a shared contract so the same rules can protect server actions and route handlers.

## Treat SEO as Product Work

Metadata, Open Graph images, descriptions, and stable URLs help the work travel well when someone shares it.
